# Introduction

BusyBox 是一个为嵌入式 Linux 系统设计的轻量级工具集，它将许多常用的 Unix 工具（如 ls、cp、mv、grep 等）
集成到一个单一的可执行文件中。BusyBox 提供了一个紧凑的环境，适用于资源受限的设备，如路由器、嵌入式系统和小型 Linux 发行版。
