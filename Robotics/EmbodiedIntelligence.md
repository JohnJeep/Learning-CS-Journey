# 1. Embodied Intelligence(具身智能)

机器人学主要依赖于数学、物理、机械工程、电子工程、计算机科学，还涉及哲学、心理学、生物学等领域。

机器人的核心是运动---受控制的编程运动。

机器人学研究的是怎样综合运用机械装置、传感器、启动器和计算机来实现人类某些方面的功能。



机器人学划分为4个主要领域

- 机械操作
- 移动
- 计算视觉
- 人工智能










线性代数在机器人中的角色。

| 应用领域      | 核心线性代数概念       | 作用                               |
| :------------ | :--------------------- | :--------------------------------- |
| **运动学**    | 齐次变换矩阵、旋转矩阵 | 计算机械臂末端位置、机器人位姿变化 |
| **动力学**    | 质量矩阵、向量运算     | 计算力、力矩、加速度，实现精准控制 |
| **视觉/感知** | 像素矩阵、特征值分解   | 图像识别、三维空间重建、避障       |
| **状态估计**  | 协方差矩阵、矩阵求逆   | 融合传感器数据，消除噪声，准确定位 |

# Kinematics(运动学)

规划与控制

机械臂通常配备有机械手或手爪’更通用的说法是末端执行器（end＿e旅ctor）,它是与周围世界的物体进行交互的载体。

’机器人位形空间与任务空间之间并非——映射的关系°机器人的工作空间（workspace）则可以定义为末端执行器所能达到的任务空间的子集。

**任务空间（task space）**：末端执行器的位姿空间。







## 3. Forward Kinematics(正向运动学)

正运动学研究物体的运动，而不考虑引起这种运动的力。

如何根据机器人的关节角度，描述每个连杆的位置和姿态，被称为正向运动学。

正向运动学中研究的内容

- 位置
- 速度
- 加速度
- 位置变量对于时间或者其他变量的高阶微分。





## 4. Inverse Kinematics(逆向运动学)

研究的是为什么要这么运动？涉及力。

根据每个连杆的位置和姿态，描述机器人关节角度，被称为逆向运动学。

# 7. ZPM
